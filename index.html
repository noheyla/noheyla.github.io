Mes réalisations

EN COURS Python

CVE Tracker — Suivi de vulnérabilités

Projet Python de suivi de vulnérabilités avec gestion des CVE, score CVSS, statut de correction, filtrage par sévérité et export de rapport. L'objectif est de simuler une petite logique de gestion de vulnérabilités exploitable dans un contexte SOC ou gouvernance.

→
TERMINÉ Wireshark

Analyse de trafic réseau

Analyse de captures réseau avec Wireshark pour observer les protocoles DNS, HTTP et TCP/IP, comprendre les échanges entre machines et identifier des événements réseau simples. Le dépôt peut inclure un compte-rendu, des captures d'écran et une synthèse d'analyse.

→
TERMINÉ C++

Gestionnaire de logs système

Application CLI orientée journalisation, traçabilité et lecture de fichiers de logs. Le projet met en avant la structure du code, le filtrage, la consultation et l'organisation d'événements système, avec une logique utile pour comprendre l'exploitation des journaux.

→
TERMINÉ PostgreSQL

Base de données — Gestion des VMs

Projet de modélisation UML et PostgreSQL autour de la gestion de machines virtuelles : schéma relationnel, contraintes, triggers et requêtes SQL. Il montre une bonne base sur la structuration de données techniques et d'environnements virtualisés.
